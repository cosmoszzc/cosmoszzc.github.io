# Personal Website

This is a static personal website prepared for GitHub Pages.

GitHub account: `cosmoszzc`

Recommended repository name: `cosmoszzc.github.io`

Expected user site URL: `https://cosmoszzc.github.io/`

## Local Preview

Open `index.html` directly in a browser.

## Deploy To GitHub Pages

1. Create a GitHub repository.
2. Push these files to the repository.
3. In GitHub, open `Settings` -> `Pages`.
4. Choose `Deploy from a branch`.
5. Select the `main` branch and `/ (root)`.
6. Save. GitHub will publish the site after the Pages build finishes.

For this user site, name the repository `cosmoszzc.github.io`; the site URL will be `https://cosmoszzc.github.io/`.

For a normal project repository, the site URL will usually be `https://<github-username>.github.io/<repository-name>/`.

## Edit Content

- Change the page title and hero name in `index.html`.
- Add a real contact email or other contact link if desired.
- Replace the placeholder sections with real bio, work, and current projects.
- If using a custom domain later, add a `CNAME` file containing only the domain name.
